vp_state_get_dirty_flags

Returns the current 32-bit dirty-flag register without clearing it.

Bit positions match the VP_FLAG_* constants in i2c_protocol.h. VP_FLAG_CHANGED (bit 0) is set whenever any field has been updated. Individual parameter bits (VP_FLAG_VOL, VP_FLAG_BAT, …) reflect which fields have changed since the last call to vp_state_clear_dirty_bits().

Returns
uint32_t Current dirty flags.
uint32_t vp_state_get_dirty_flags (void)
 
vp_state_clear_dirty_bits

Atomically clears the specified bits from the dirty-flag register.

Called by i2c_bridge after the RPi has successfully fetched a parameter. If clearing the last parameter bit also makes VP_PARAM_BITS_MASK == 0, VP_FLAG_CHANGED is automatically cleared as well.

Parameters
maskBitmask of VP_FLAG_* bits to clear.
void vp_state_clear_dirty_bits (uint32_t mask)
